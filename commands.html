Command List

blargbot is equipped with a number of miscellaneous commands. By default, blargbot uses the command prefix '!'. This can be changed with the setprefix command. It will also respond to it's name, 'blargbot'.

Some commands have permission requirements. In order to satisfy them, you need to have a role with the name detailed in the Permissions section.

General commands

cat

Usage: cat
Permissions: none

Displays a picture of a cat, taken from random.cat

danbooru

Usage: danbooru <tags>
Permissions: 'Porn Summoner' (if not safe)

Displays three images obtained from danbooru.donmai.us using the provided tags. You can use up to 2 tags at a time. Result have to possibility of being NSFW. If the current channel is not designated as NSFW, a user needs the role by the name of "Porn Summoner" in order to use this command without including the 'rating:safe' tag.

e621

Usage: e621 <tags>
Permissions: 'Porn Summoner' (if not safe)

Displays three images obtained from e621.net using the provided tags. You can use up to 6 tags at a time. Result have to possibility of being NSFW. If the current channel is not designated as NSFW, a user needs the role by the name of "Porn Summoner" in order to use this command without including the 'rating:safe' tag.

econ

Usage: econ <from> <to> <amount>
Permissions: none

Converts currency using recent rates. Example:

User> blargbot econ USD CAD 1
blargbot> @User, 1.0 USD is equivalent to X.X CAD

Currency codes are not case sensitive.

help

Usage: help (command)
Permissions: none

Returns a list of built-in commands and custom commands. If a command is specified, it will return a description of that command instead.

info

Usage: info
Permissions: none

Returns information about the bot.

insult

Usage: insult (name)
Permissions: none

Generates an insult, direct at whatever name is supplied. If a name isn't supplied, defaults to 'Your'

invite

Usage: invite
Permissions: none

Gives a link to invite me to your discord, and gives another link to invite you to my discord!

join

Usage: join
Permissions: none

Does the same thing as invite.

ping

Usage: ping
Permissions: none

roll

Usage: roll (dice) (modifiers)
Permissions: none

Rolls dice and adds a modifier. You can specify the number of dice and how many sides they have with the notation XdY, where X is number of dice and Y is number of sides. If no dice are specified, it defaults to 1d20.

suggest

Usage: suggest <suggestion>
Permissions: none

Sends a suggestion to my discord. Thank you for the feedback! It's very important to me.

tag

Usage: tag [<name> | create | edit | delete | raw | author | search | list | help]
Permissions: none

Tags are like public custom commands. You can create them on one guild, and use them on another. Anyone is capable of making tags. Tags use a tagging system, so they can can range from simple to complex. See the tag page for more details. 

tag <name>

Executes a tag with the given name. 

tag create <name> <content>

Creates a tag with the given name and content. The name must be unique. 

tag edit <name> <content>

Edits an existing tag. You must own it to edit it!

tag delete <name>

Deletes the specified tag. You must own it to delete it!

tag raw <name>

Outputs the raw tag

tag author <name>

Tells you who made the specified tag

tag search <name>

Searches for tags with given name

tag list

Lists all tags

tag help

Get's basic tag help.

user

Usage: user (id)
Permissions: none

Gets information about the specified user.

xkcd

Usage: xkcd (number)
Permissions: none

Gives you an xkcd comic. If a number isn't specified, it gives you a random one.

Admin Commands

ccommand

Usage: ccommand <command name> (command content)
Permissions: 'Bot Commander'

Creates a simple custom command. Once created, whenever you say the command name, blargbot will respond with the command content. For example:

User> blargbot ccommand test This is a test command.
blargbot> Created command test
User> blargbot test
blargbot> This is a test command.

You can also use a %REPLY tag to make blargbot reply to the user.

input: Hello, %REPLY.
output: Hello, @User

To remove a custom command, simply do ccommand without command content. For example:

User> blargbot ccommand test
blargbot> Removed command test
User> blargbot test
*no response from blargbot*

Finally, custom commands take precedent over all other commands. As such, you can use it to overwrite commands, or disable them entirely. If the command content is "null" (without the quotations), blargbot will have no output whatsoever, allowing you to disable any built-in command you wish. You cannot overwrite the 'ccommand' command.

greet

Usage: greet (greeting)
Permissions: 'Bot Commander'

Sets a greeting for any new user who joins your discord. You can use a %USER tag to get the user's name. For example:

User> blargbot greet Welcome, **%USER**. Please read #rules.
blargbot> Greeting set. Simulation: Welcome, **User**. Please read #rules.

nsfw

Usage: nsfw
Permissions: 'Bot Commander'

Designates the current channel as NSFW, allowing any user to do a NSFW command with the 'Porn Summoner' role.

purge

Usage: purge
Permissions: 'Bot Commander'

Deletes all the messages within 500 messages the bot has said in the current channel

setprefix

Usage: setprefix <prefix>
Permissions: 'Bot Commander'

Sets the command prefix for the guild. You can set it to anything that is one word long (no spaces). Set it to 'null' to disable it entirely and use only 'blargbot' as a prefix.

tidy

Usage: tidy (amount)
Permissions: 'Bot Commander'

Cleans a number of messages, defaulting to 25.
